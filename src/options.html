Keyboard shortcuts
  • [Ctrl(Cmd) + B] Focus sidebar
  • [Ctrl(Cmd) + K] Navigate to "/library"
  • [Ctrl(Cmd) + Shift + K] Navigate to "/spaces"
  • [Up/Down & Enter on "/library" search box] Navigate through search results
  • [Up/Down/Left/Right & Enter on "/spaces"] Navigate through spaces
  • [Shift + Enter] Add Markdown list symbols and indentation along with line breaks
  • [Tab] Indent the current line (Selection operation support)
  • [Shift + Tab] Outdent the current line (Selection operation support)
  • [Ctrl(Cmd) + ArrowDown] Switch to next search mode
  • [Ctrl(Cmd) + ArrowUp] Switch to previous search mode
  • [Ctrl(Cmd) + Shift + ArrowDown] Switch to the next model
  • [Ctrl(Cmd) + Shift + ArrowUp] Switch to the previous model
  • [Ctrl(Cmd) + Shift + .] Toggle Web search source
  • [Shift + Click the Copy button] Copy without citation links
  • [Click the Copy button] Copy with citation links (Normal Copy)
Optimization (Always enabled)
  • [Esc] Sets focus on the scrollable container instead of the textarea
  • [Ctrl+V] Prevents the cursor from jumping to the end
  • When navigate to "/library", Sets focus on the search box
Mermaid preview
When enabled, the Mermaid code block is automatically detected and a preview button is displayed.
When set to "auto", either "default" or "dark" is selected according to the OS theme.
Table of Contents (TOC) (Experimental)
When enabled, displays a floating table of contents on the right side of search results that allows quick navigation between sections.
